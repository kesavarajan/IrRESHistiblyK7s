HOW TO ADD GALLERY PHOTOS (two ways)

A) Easiest - numbered:
   Put files here named 1.jpg, 2.jpg, 3.jpg ... (jpg/jpeg/png/webp).
   Leave list.json as []  (empty).

B) Any file names you like:
   Edit list.json with your file names, e.g.
   ["engagement.jpg", "beach-sunset.png", "temple.jpg"]
   Then just drop those exact files in this folder.

Note: iPhone .heic files do NOT show in browsers - save/convert them to .jpg first.
